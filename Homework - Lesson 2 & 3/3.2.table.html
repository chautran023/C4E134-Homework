Table elements I've learnt

Element Description
The root element
table which is the main container
td used to create data cells
tr represents the rows
Table Heading
th add a table heading
thead used to create a separate header for the table
Other attributes
scope identifies whether a cell is a header for a column, row, or group of columns or rows
caption provide a description about the data presented in a table
colspan and rowspan stretches 2 or more cells across a table
tbody it indicates the main body of the table
tfoot used to create a separate footer for the table
